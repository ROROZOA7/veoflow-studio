# Images Directory

This directory stores all images, screenshots, and thumbnails generated by the application.

## Contents

- **Screenshots**: Debug screenshots from Flow automation and error detection
- **Thumbnails**: Scene thumbnails (if implemented)
- **Test Images**: Screenshots from test runs

## Configuration

The images directory path is configured in:
- `app/config.py` - `IMAGES_PATH` setting (default: `./images`)
- `veoflow.config.json` - `paths.images` setting

## Usage

All screenshots are saved using the `get_screenshot_path()` function in `app/services/flow_controller.py`, which automatically saves to this directory.
